# 🏨 Projet de gestion d'hôtel

Ce projet est une application complète de gestion d'hôtel utilisant **MySQL**, **SQLite**, **Python** et **Streamlit**.  
Il permet la consultation et la gestion de réservations, de clients et de chambres d’hôtel via une interface web simple.

---

## 📁 Structure du projet

hotel_project_streamlit/
├── interface.py # Application Streamlit
├── hotel_db.sqlite # Base de données SQLite
├── fichier 2.sql #Script SQL pour créer la BDD dans MySQL
├── Requetes_SQL_Algebre_Hotel.pdf # Requêtes en algèbre relationnelle
├── README.md 


---

## ⚙️ Fonctionnalités de l’application

1. **Consultation des réservations**
2. **Affichage des clients**
3. **Liste des chambres disponibles pour une période donnée**
4. **Ajout de nouveaux clients**
5. **Ajout de nouvelles réservations**

---

## 📌 Partie 1 : Base de données

- Création de la base de données avec MySQL
- Remplissage des tables `Hotel`, `Chambre`, `Client`, `Type_Chambre`, `Prestation`, `Réservation`, `Évaluation`
- Écriture des requêtes SQL + algèbre relationnelle (voir le fichier PDF joint)

---

## 📌 Partie 2 : Application Streamlit

- Interface web simple
- Connexion à SQLite (`hotel.db`)
- Consultation dynamique des données
- Formulaires pour ajouter clients/réservations

---
